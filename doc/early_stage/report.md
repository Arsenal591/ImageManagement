#图片分享网站 前期文档  

##一、概述
我们利用Django框架搭建了一个图片分享网站，支持用户注册、上传与社交，并可以对图片进行各种编辑处理。  

##二、功能简述  
###1. 用户管理  
###2. 图片管理
1. 图片的上传  
	* 单独上传  
	* 批量上传  
2. 图片的处理
	* 灰度图
	* 二值化
	* 高斯模糊
	* 缩放
	* 旋转
	* 剪裁
3. 图片的信息标注
	* 自动标注类别
	* 用户手动添加类别
	* 点赞
4. 图片的展示
	* 按照上传时间
	* 按照上传用户
	* 按照赞数
	* 按照类别
	* 按照相似度（以图搜图）
